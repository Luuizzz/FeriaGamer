🎮 Juego narrativo de investigación psicológica enfocado en el impacto del cyberbullying, el acoso y las relaciones digitales.

El jugador explora un entorno interactivo mientras utiliza el celular del protagonista para descubrir conversaciones, evidencias y decisiones que afectan directamente el desarrollo de la historia. El sistema combina Unity 2D con interfaces web integradas para crear una experiencia inmersiva basada en diálogos dinámicos, empatía y consecuencias emocionales.

⚠️ El juego aborda temas sensibles relacionados con acoso digital, manipulación social y salud emocional.

Tecnologías utilizadas:

* Unity 2D
* C#
* HTML / CSS / JavaScript
* WebView Integration
* Sistema de diálogos condicionales con Reflection

📌 Proyecto académico / experimental orientado a narrativa interactiva y diseño de sistemas.
